This widget is intended to be a supplemental tool, to enable Ambassador users the ability to share their sermons on their WordPress websites.

== Description ==

An ambassador sermon account is required in order to secure an actual sermon feed from AmbassadorApp.com.

The admin panel setup will request 2 pieces of information that can only be attained from your Ambassador Admin panel.  The first piece is the embed link, which can be copied from you Tools > Embeddable Widget. The text between the src parenthesis will need to be copied and pasted into the annotated text box. The second piece is the link to the user's subscription page, which will allow viewers to subscribe to a church's sermon email list. This link can be copied from the Members > Member Subscribe Page url bar.

== Installation ==

1. Upload ambassador-sermon-widget.php to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag and drop the plugin to your desired sidebar or footer area in the Appearance > Widget panel.
